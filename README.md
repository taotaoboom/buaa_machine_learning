## 关于机器学习成绩组成
截至2026.1.12最新情况：机器学习现在是全校统一考试，成绩组成为平时成绩（签到+个人作业+小组作业）：期末考试（闭卷）=1:1，其中平时的作业是布置一些任务（如香蕉成熟分类，虚假新闻检测），依据性能的排行榜赋分，另外还要交自己代码实现的报告。考试题型为10到简答题，无选择、判断题。
